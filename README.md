# Liquid Player

Медиа-плеер на Python с дизайном в стиле iOS 26 «Liquid Glass» и UI на 120 FPS.

## Что внутри

- **Liquid Glass UI** — анимированный градиентный фон с плавающими цветными блобами,
  фростед-стеклянные карточки со скруглениями, бликами и тонкой обводкой.
- **Кастомные кнопки** — круглые «капельные» кнопки с акцентным play и тактильной
  анимацией нажатия.
- **Кастомный слайдер** — стеклянный жёлоб, светящаяся заливка, плавающий knob с
  ореолом-glow.
- **Визуализатор** — 32 синусоидальных столбика, плавно интерполирующих уровень.
- **120 FPS** — все анимации обновляются через `QTimer` с `PreciseTimer` на ~8 мс.
- **Аудио** — воспроизведение через `QMediaPlayer` (mp3/wav/flac/ogg/m4a/aac).

## Запуск

```bash
pip install -r requirements.txt
python media_player.py
```

Нажми `➕`, чтобы добавить треки, потом ▶ — играть, ⏮/⏭ — перелистывать.
Перетягивай прогресс, чтобы перемотать; нижний слайдер — громкость.

## Замечания по 120 FPS

`TARGET_FPS = 120` в `media_player.py` задаёт интервал обновления (~8 мс).
Реальная частота кадров будет ограничена частотой обновления монитора — на 60 Гц
Qt будет ронять лишние кадры на vsync, на 120/144 Гц всё проиграется честно.
Если хочется снизить нагрузку — поставь `TARGET_FPS = 60`.
