# xd launcher

Minecraft-лаунчер на Python в духе Prizm Launcher. PyQt6 + minecraft-launcher-lib.

## Возможности

- **Версии в один клик:** 1.12.2, 1.16.5, 1.20.1 (плюс всё, что уже установлено).
- **Кнопка «⟳ Обновить»** — перезагружает список версий.
- **Кнопка «📁 Папка»** — открывает папку с игрой.
- **RAM-слайдер** — определяет общий объём ОЗУ через `psutil` и даёт выбрать,
  сколько отдать игре.
- **Разрешение игры** — по умолчанию 925 × 350, как ты и просил.
- **Аргументы Java** — своё поле для произвольных JVM-флагов.
- **Оптимизированные аргументы** — рабочий набор G1GC-флагов, можно
  редактировать и сбрасывать к дефолту.
- **Команда-обёртка** — например `prime-run`, `gamemoderun`, `optirun` —
  лаунчер подставит её перед `java`.
- **Авто-установка Java** — Adoptium JRE качается под нужный мажор
  (Java 8 для 1.12.2/1.16.5, Java 17 для 1.20.1). Можно указать свою Java.
- **GPU-выбор** — дискретная/встроенная: ставит `__NV_PRIME_RENDER_OFFLOAD`,
  `DRI_PRIME`, `SHIM_MCCOMPAT`, при наличии `prime-run` использует его как обёртку.
- **Свой фон** лаунчера (jpg/png).
- **Modrinth-кнопка** — открывает modrinth.com. Кнопка «Установить файл…»
  сама раскладывает `.jar` в `mods/`, шейдеры (по имени) в `shaderpacks/`,
  остальные `.zip` в `resourcepacks/`.
- **Аккаунты** — добавляй сколько хочешь ников, выбирай активный в шапке.
- **Лог запуска** — отдельная вкладка.

## Установка

```bash
pip install -r requirements.txt
python xd_launcher.py
```

## Где живут данные

Лаунчер хранит конфиг, аккаунты, игру и Java-рантаймы в:

- Linux: `~/.xd-launcher/`
- macOS: `~/Library/Application Support/xd-launcher/`
- Windows: `%APPDATA%\xd-launcher\`

Папка игры — `<root>/minecraft/`; туда же лаунчер кладёт `mods/`,
`resourcepacks/`, `shaderpacks/`.

## Заметки

- Аккаунты офлайновые — UUID считается из ника (как у Prizm/MultiMC в offline mode).
- Для подключения к лицензионным серверам нужны premium-аккаунты Microsoft,
  они здесь не поддержаны.
- Если у тебя в системе уже стоит Java правильной версии — можно снять галку
  «Скачивать Java автоматически» и указать путь вручную.
