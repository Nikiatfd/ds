# XD Launcher

Лёгкий Python-лаунчер Minecraft в духе **Prizm Launcher**. Запускает Vanilla
версии **1.12.2 / 1.16.5 / 1.20.1** (и любые другие, которые ты добавишь
вручную), скачивает Java под версию, умеет переключать видеокарту,
команду-обёртку и оптимизированные JVM-аргументы.

## Запуск

```bash
pip install -r requirements.txt
python xd_launcher.py
```

При первом запуске данные сохраняются в `~/.xd_launcher/` (профиль игры,
runtime Java, конфиг).

## Возможности

- 🎮 **Версии**: 1.12.2, 1.16.5, 1.20.1 предустановлены; можно добавлять свои
  через «+ Добавить версию» и обновлять список кнопкой «⟳ Перезагрузить».
- 🧠 **RAM**: ползунок ограничен реальным объёмом ОЗУ (определяется
  через `psutil`), значение пробрасывается как `-Xmx`/`-Xms`.
- 📁 **Папка игры**: кнопка открывает `~/.xd_launcher/minecraft/` в файловом
  менеджере.
- ☕ **Java**: автоматическая установка нужного JRE через
  `minecraft-launcher-lib` *или* свой путь к `java`.
- 📐 **Разрешение**: по умолчанию **925 × 350**, меняется в настройках.
- ⚙️ **Аргументы**: поле произвольных JVM-аргументов + готовый набор
  оптимизированных (G1GC, MaxGCPauseMillis=50 …).
- 🧩 **Команда-обёртка**: например `gamemoderun`, `prime-run`, `optirun …` —
  подставляется перед командой Minecraft.
- 🎛 **Видеокарта**: NVIDIA (`__NV_PRIME_RENDER_OFFLOAD=1`), AMD (`DRI_PRIME=1`),
  встроенная (`DRI_PRIME=0`).
- 🖼 **Кастомный фон**: PNG/JPG → главная панель.
- 🌐 **Modrinth**: открывает modrinth.com и сортирует скачанные файлы из
  выбранной папки в `mods/`, `resourcepacks/`, `shaders/`.
- 👤 **Аккаунты**: оффлайн-ники + вход через **Ely.by** (`authserver.ely.by`).
  При запуске под Ely.by-аккаунтом лаунчер сам качает
  `authlib-injector.jar` и подмешивает
  `-javaagent:authlib-injector.jar=https://authlib-injector.ely.by`,
  чтобы заходить на лицензионные сервера под зарегистрированным на Ely.by ником.

## Структура данных

```
~/.xd_launcher/
├── config.json        # настройки
├── minecraft/         # игра (профили, mods, resourcepacks, shaders…)
└── runtime/           # авто-скачанная Java
```

## Зависимости

- `PyQt6` — GUI
- `minecraft-launcher-lib` — установка/запуск Minecraft и Java runtime
- `psutil` — определение ОЗУ
- `requests` — сетевые запросы (используется зависимостями)
